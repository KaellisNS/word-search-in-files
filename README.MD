Необходимо реализовать компонент для поиска ключевого слова в файлах.
Текстовый файл содержат слова, которые разделены пробелами. 
Знаки препинания отсутствуют. Если слово не найдено, то возвращаем nil для списка файлов. 
Если при поиске произошла ошибка, то возвращаем ошибку и nil для списка файлов.
Для реализации поиска разрешается использовать только стандартную библиотеку.
Также необходимо дополнить тесты для следующих случаев: слово не найдено; при обработке файла возникла ошибка.
Решение должно быть корректным и возвращать правильный результат.
Комментарии и пояснения к коду приветствуются.

Плюсом будет.
Реализация параллельного поиска по файлам. Поиск слова за O(1).
Реализация rest-API для работы с файлами: загрузка файла на сервер, просмотр списка файлов, поиск ключевого слова в файлах. 
Для построения веб-API можно использовать стандартную библиотеку или один из веб-фреймворков (Echo, gin и т.д.). Тесты для API будут плюсом.
